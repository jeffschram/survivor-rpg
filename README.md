# survivor-rpg
AI-driven RPG version of "Survivor" with an AI GM.

## Run locally

1. Install dependencies:
   `npm install`
2. Create a `.env` file in the project root:
   `OPENAI_API_KEY="your-key"`
   `OPENAI_MODEL="gpt-4o-mini"` (optional)
3. Start the server:
   `npm start`
4. Open `http://localhost:3000`
